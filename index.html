Introduction to HTML by Daria

Doctype

Description

A bit of code that you place at the top of HTML file to tell the browser exactly which version of HTL you are using.

Example of the Code

<!doctype html>

html

Description

(HyperText Markup Language)is the actual skeleton of the webpage-it`s a markup language.

Example of the Code

<html> </html>

head

Description

Is the brain of the webpage, it contains all the information about the page that the browser needs to know.

Example of the Code

<head> </head>

body

Description

Is the entire physical presence, everything you see, click or interact with on a webpage lives inside the body.

Example of the Code

<body> </body>

title

Description

It defines the name of the webpage, it never actually appears on the page itself.

Example of the Code

<title>Introduction to HTML by Daria</title>

p

Description

It is the basic building block for any block of text on a webpage.

Example of the Code

<p>This is a paragraph</p>

Working Example

This is a paragraph

span

Description

It is a container used to hook on specific words so you can style them later with CSS

Example of the Code

<p>This is a <span style="color:red" > inline span. </span> This is outide the span

Working Example

This is a inline span. This is outide the span

div

Description

Is a big empty cardboard box.

Example of the Code

<div>
<p> This is a Div it divides our page </p>
</div>
<p>This is a paragraph utside the div</p>

Working Example

This is a Div it divides our page

This is a paragraph outside the div

formatting elements

Description

Are tools you use to change the appearance of specific words within the blocks.

Example of the Code

<p>This is my <strong>first</strong> HTML document</p>
<p>This is my <em>first</em> HTML document</p>

Working Example

This is my first HTML document

This is my first HTML document

They help organise the content so it is easy for users to scan.

Working Example

Description

quote and blockquote

Description

The quote is used for short quotes that sit right inside a regular paragraph while the blockquote is used for long-form quotes that come from another source.

Example of the Code

<p>Some famous poet wrote <q> To be or not to be</q></p>

<blockquote>
<p>The present is theirs, the future, for which I really worked, is mine.</p>
</blockquote>

Working Example

Some famous poet wrote To be or not to be

The present is theirs, the future, for which I really worked, is mine.

abbreviations

Description

Is a digital sticky note for a shortened word.

Example of the Code

<p>The <abbr title="Limerick and Clare Education and Training Board">LCETB</abbr> was established on july 1, 2013.</p>

Working Example

The LCETB was established on july 1, 2013.

<meta name="description" content="assignment for Web Authoring">
